CLASS FEATURES

Many, but not all, 직업 특성 continue to accumulate after 20th 레벨. The following guidelines describe how the 에픽 직업 성장.

• A character’s base save 보너스 and base 공격 보너스 don’t increase after 20th 레벨. Use 표: 에픽 내성굴림과 에픽 공격 보너스 to determine the character’s 에픽 보너스 on 내성 굴림 and 공격.

• Characters continue to gain 히트 다이스 and skill points as normal beyond 20th 레벨.

• For spellcasters, 시전자 레벨 continues to increase after 20th 레벨. However, 주문 per day don’t increase after 20th 레벨. The only way to gain additional 주문 per day (other than 보너스 주문 gained from a high ability score) is to select 향상된 주문 용량 에픽 feat.

• The powers of familiars, special 탈것 소환, and fiendish servants continue to increase as their masters gain 레벨.

• Any 직업 특성 that increase or accumulate as part of a repeated pattern also continues to increase or accumulate after 20th 레벨 at the same rate.  An exception to this rule is any 보너스 feat granted as a class feature. If a character gets 보너스 특기 as part of a class feature, these do not increase with 에픽 레벨. Instead, these classes get 보너스 특기 at a different rate (described in each 에픽 class 묘사).

• In addition to 직업 특성 retained from nonepic 레벨, each class gains 보너스 feat every two, three, four, or five 레벨 after 20th. This augments each class’s progression of 직업 특성, because not all classes otherwise improve 직업 특성 after 20th 레벨. A character must select these 특기 from the list of 보너스 특기 for that class. These 보너스 특기 are in addition to the feat that every character gets every three 레벨. The character isn’t limited to selecting from the class list when selecting these 특기.

• Characters don’t gain any new 직업 특성, because there aren’t any new 직업 특성 described for these 레벨. 직업 특성 with a progression that 감속화 or stops before 20th 레벨 and features that have a limited list of options do not improve as a character gains 에픽 레벨. Likewise, 직업 특성 that are gained only at a single 레벨 do not improve.
